TripleSpec Slit Mapping

Dec 25th, 2012

Working on a TripleSpec Slit Mapping project on github. In its previous iteration, we assumed a fixed offset of 0.5” per step, and no pointing errors. This method uses the slit viewer to derive a pointing solution using astrometry.net, then apply that solution to each slit position to make a map.
